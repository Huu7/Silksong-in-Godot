#How do use
Clone this repository to be able to access everything within it and make any modifications to your liking in Godot

If you just want to download and play the game:
#Windows
[Click here](https://github.com/DyedHue/Silksong-Godot/raw/refs/heads/main/Release/Silksong%20Godot%20Windows.zip) to directly download the ZIP file for playing the game on Windows
or you can manually download all the files inside the [Release/Windows](/Release/Windows) folder

#Linux
[Click here](https://github.com/DyedHue/Silksong-Godot/raw/refs/heads/main/Release/Silksong%20Godot%20Linux.zip) to directly download the ZIP file for playing the game on Linux
or you can manually download all the files inside the [Release/Linux](/Release/Linux) folder
